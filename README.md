## ⚒️ Technologies

Project is created with:

## 📑 Descriptions

- Use React hooks
- Display times and leftover bombs on the top
- Reset game by clicking the smile emoji
- CSS Grid / Flexbox
- SASS/SCSS mixins / variables
- typescript Enum type / interface
- typescript interface
